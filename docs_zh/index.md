## 📚 项目描述

**Code Review GPT Gitlab** 是一个利用大模型（包括但不限于 GPT、DeepSeek 等）帮助开发团队在 GitLab 上进行自动化代码审查，提升研发效能的智能工具。

### 核心特点 ✨

- 🤖 **智能代码审查** - 通过大模型自动分析代码质量、潜在问题和改进建议
- 🔗 **GitLab 深度集成** - 专为 GitLab 定制，通过 Webhook 自动触发代码审查
- 🎯 **多模型支持** - 支持接入多种大模型（GPT、DeepSeek 等），包括私有化 LLM
- 📢 **多渠道通知** - 支持钉钉、飞书、微信等多种消息通知方式
- ⚙️ **高度可定制** - 支持自定义 Prompt 提示词，灵活配置审查规则和逻辑
- 🚀 **Multi-Agent 架构** - 正在开发多 Agent 协同工作模式，提升审查质量
- 🔒 **代码安全检测** - 能够识别代码安全问题，提升代码安全性

## 🎯 主要功能

### 1. 自动化代码审查
- 当 GitLab 项目创建 Merge Request 时，自动触发代码审查
- 分析代码变更，提供详细的审查报告和建议
- 支持多种编程语言的代码审查

### 2. 灵活的模型配置
- 支持配置多个大模型服务
- 可以针对不同项目使用不同的模型
- 支持私有化部署的 LLM 服务

### 3. 智能通知系统
- 自动将审查结果发送到配置的渠道（钉钉、飞书、微信等）
- 支持自定义通知格式和内容
- 实时反馈审查进度和结果

### 4. 项目管理
- 统一管理多个 GitLab 项目
- 查看历史审查记录和统计信息
- 支持项目级别的配置管理
