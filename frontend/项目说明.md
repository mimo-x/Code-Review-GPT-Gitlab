### 4. TypeScript 类型安全

- 折线图(审查趋势)

- 渐变色配置

## 🚀 如何运行

### 1. 安装依赖
```bash
cd frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

访问: http://localhost:3000

### 3. 构建生产版本
```bash
npm run build
```
