为研发团队量身定制

Code-Review-GPT-Gitlab 不仅仅是一个机器人，它是您团队中不知疲倦的代码质量守护者。

Gitlab 无缝集成

专为 Gitlab 打造，通过 Webhook 自动触发。支持 Merge Request 事件，评论直接反馈在代码行间。

Multi-Agent 协同

引入多智能体架构，模拟真实团队评审流程。不同 Agent 关注安全、性能、风格等不同维度。

多模型灵活切换

支持 GPT-4, DeepSeek, Claude 等主流大模型。根据预算和需求，灵活配置最适合的 AI 引擎。

私有化安全部署

数据不出域，核心代码安全无忧。支持接入私有化部署的 LLM 模型，满足企业级合规要求。

高度可配置

自定义 Prompt 提示词，定制评审标准。支持忽略特定文件、设置触发关键词等细粒度控制。

效能洞察报告

不仅仅是 Review，更提供研发效能分析。生成详细的评审报告，帮助团队持续改进代码质量。

简单三步，即刻启用

无需复杂的配置，通过 Docker Compose 即可在几分钟内完成私有化部署。

1

配置 Webhook

在 Gitlab 项目设置中添加 Webhook，指向您的服务地址。

2

启动服务

使用 Docker Compose 一键启动后端服务。

cp .env.example .env docker compose up -d
3

自动评审

提交 MR，AI 助手将自动进行评审并发表评论。

常见问题

支持哪些 Gitlab 版本？
目前支持 Gitlab CE/EE 13.0 及以上版本。只要您的 Gitlab 实例支持 Webhook 功能即可接入。
如何接入私有化模型？
后端支持 OpenAI 兼容接口。如果您使用 Ollama、LocalAI 或 vLLM 等工具部署了私有模型，只需在 .env 文件中配置对应的 API Base URL 即可。
是否收费？
本项目完全开源（MIT License），您可以免费使用、修改和分发。使用的 LLM 模型费用由您自行承担（如调用 OpenAI API）。
